About TaskFlow Manager

TaskFlow Manager is a web-based task management system developed using Django and SQLite.

It helps users create, organize and monitor tasks efficiently through an intuitive interface.

Features

  • 📋 Add New Tasks
  • 📊 View All Tasks
  • ⚡ Track Task Priority
  • ✅ Monitor Task Status
  • 💾 Store Data Using SQLite Database
